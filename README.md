# Data-Analysis-Project - Malaria Survey Data in Nigeria (DHS Survey Data)

One of the major serious health problems in Nigeria is Malaria, although preventable and treatable. This spans across the west african region.

Africa still bears over 80% of the global malaria burden, and Nigeria acconts for about 29% of it.

Our project (notebok) shall rely on Nigeria Malaia Indicator Survey dataset to perfom some descriptive analysis, visualization and machine learning predictions.
